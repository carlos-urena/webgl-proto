Simple WebGL 3D mesh viewer.

A simple prototype for rendering meshes with WebGL+GLSL. This is work in progress. Source code is available at this Github repository: github.com/carlos-urena/webgl-proto. Drag (right mouse button) or swipe (on mobile devices) to rotate the camera. Use mouse wheel to zoom in or out. You can also drag & drop a PLY file.
